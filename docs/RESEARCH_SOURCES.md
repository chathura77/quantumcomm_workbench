# Research sources and reference map

Use these as starting references for the app's Resources section and explanatory copy. Verify details before adding precise claims.

## Simulators and frameworks

- NetSquid: discrete-event quantum network simulator. URL: https://netsquid.org/
- SeQUeNCe: Simulator of QUantum Network Communication. URL: https://github.com/sequence-toolbox/SeQUeNCe
- QuISP: Quantum Internet Simulation Package. URL: https://aqua.sfc.wide.ad.jp/quisp_website/
- QuNetSim: quantum network simulation framework. URL: https://github.com/tqsd/QuNetSim
- SimulaQron: distributed quantum internet simulator. URL: https://github.com/SoftwareQuTech/SimulaQron
- NetQASM and SquidASM: application-level quantum network programming and simulation. URL: https://github.com/QuTech-Delft
- QKDNetSim: QKD network simulation extension associated with NS-3. URL: https://www.qkdnetsim.info/

## Standards and guidance

- ETSI ISG QKD publications. URL: https://www.etsi.org/technologies/quantum-key-distribution
- ETSI GS QKD 014: REST-based key delivery API. URL: https://www.etsi.org/deliver/etsi_gs/QKD/001_099/014/
- ITU-T Y.3800 series on quantum key distribution networks. URL: https://www.itu.int/rec/T-REC-Y.3800
- ITU-T Y.3803 on QKD network key management. URL: https://www.itu.int/rec/T-REC-Y.3803
- NIST post-quantum cryptography project. URL: https://csrc.nist.gov/projects/post-quantum-cryptography
- NSA QKD/QC guidance. URL: https://www.nsa.gov/Cybersecurity/Quantum-Key-Distribution-QKD-and-Quantum-Cryptography-QC/
